# 📚 カリタス中学校 AI学習ツール

カリタス中学校の体系数学・Progress 21に準拠したAI搭載学習システム

## 🌟 特徴

### 🤖 AI問題生成
- **OpenAI GPT-4** による高品質な数学問題生成
- **段階的詳細解説** - 7つのステップで完全理解
- **思考力重視** - 暗記ではなく理解を促進

### 📚 問題プールシステム
- **即座に取得** - プールから瞬時に問題配信
- **AI + プール** - 両方のメリットを活用
- **効率的学習** - 待機時間なしの連続学習

### 🎯 完全準拠
- **体系数学** - カリタス中学校カリキュラム対応
- **Progress 21** - 英語教材完全準拠
- **学年別・分野別・難易度別** - 細かい設定対応

## 🛠️ 技術スタック

### フロントエンド
- **React 18** - モダンなUI開発
- **Vite** - 高速ビルドツール
- **Tailwind CSS** - ユーティリティファーストCSS
- **カスタムフック** - 再利用可能なロジック

### バックエンド
- **Node.js + Express** - API サーバー
- **OpenAI API** - AI問題生成
- **JSON ファイルストレージ** - 問題プール管理

## 🚀 セットアップ

### 必要要件
- Node.js 16.0.0以上
- npm または yarn
- OpenAI APIキー

### インストール

```bash
# リポジトリをクローン
git clone <repository-url>
cd caritas-study-app

# 依存関係をインストール
npm install

# 開発サーバー起動
npm run dev
```

### APIサーバー設定

1. `server.js` の OpenAI APIキーを設定
2. サーバーを起動

```bash
# サーバー起動（別ターミナル）
node server.js
```

### 環境設定

```bash
# .env ファイル作成
VITE_API_URL=http://localhost:3001
OPENAI_API_KEY=your_openai_api_key_here
```

## 📖 使用方法

### 🧮 数学学習

1. **学習設定**
   - 学年選択（中1〜高1）
   - 分野選択（全分野または特定単元）
   - 難易度選択（基礎〜発展）

2. **問題取得方法選択**
   - 📚 **問題プール**: 即座に取得
   - 🤖 **AI生成**: オーダーメイド問題

3. **学習実行**
   - 問題を解く
   - 🧠 **AI詳細解説** で理解を深める
   - 次の問題に進む

### 📚 英単語学習

1. **設定**
   - 学年・レベル選択
   - Progress 21準拠

2. **学習**
   - AI生成英単語
   - 発音・語源・例文
   - 類義語学習

### ⚙️ 管理機能

1. **AI問題一括生成**
   - 5〜20問を一度に生成
   - 学年・分野・難易度指定

2. **問題プール管理**
   - 生成問題をプールに追加
   - 統計情報表示
   - 一括追加機能
